#  WhatsApp Chat Analyzer – ML

A Streamlit-based web application that analyzes WhatsApp chat data and provides meaningful insights using data analysis, NLP, and basic machine learning techniques.



##  Features

- Upload WhatsApp chat `.txt` file
- User-wise and overall chat analysis
- Total messages, words, media, and links count
- Monthly and daily chat timelines
- Most active days and months
- Weekly activity heatmap
- Most active users analysis
- Word cloud generation
- Most common words visualization
- Emoji usage analysis
- Sentiment analysis (Positive, Neutral, Negative)
- Message classification
- User personality insights (overall)



##  Tech Stack

- Python  
- Streamlit  
- Pandas  
- Matplotlib  
- Seaborn  
- NLTK

##  Export WhatsApp Chat

1. Open WhatsApp  
2. Select a chat → Tap on **More options (⋮)**  
3. Click **Export chat**  
4. Choose **Without media**  
5. Save the `.txt` file  



##  How to Run the Project

### Clone the Repository
 
git clone https://github.com/SANJAY0103-tech/Whatsapp_chat_Analyser-ML.git
cd Whatsapp_chat_Analyser-ML


### Install Dependencies

pip install streamlit pandas matplotlib seaborn nltk wordcloud emoji


### Run the Application

 
streamlit run app.py
